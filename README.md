# 惊蛰 · Jingzhe
一个简约清新风格的响应式静态网页项目，包含首页展示、用户登录与注册功能

## 功能特性
- 响应式页面布局，适配电脑、平板、手机等不同设备
- 完整首页展示模块
- 用户登录页面
- 简洁的视觉设计

## 技术栈
- HTML5
- CSS3

## 项目结构

Jingzhe/
├── index.html       # 首页
├── login.html       # 登录页面
├── register.html    # 注册页面
├── css/             # 样式文件
├── images/          # 图片资源
└── README.md        # 项目说明
1. 克隆或下载本项目
2. 直接打开 `index.html` 即可在浏览器中预览

- 结构清晰，易于理解和二次修改
- 页面美观，适合作为前端入门作品
- 包含完整的页面体系：首页 + 登录 + 注册
- 纯前端实现，无需后端环境
