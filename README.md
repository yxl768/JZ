# 惊蛰 · Jingzhe(练着玩的...)

一个简约清新风格的响应式静态网页项目，包含首页展示、用户登录与注册功能。

## 项目介绍
「惊蛰」是为该公司做的的前端展示项目，采用现代化的页面布局与简洁视觉风格，从基础页面逐步扩展到完整的用户交互模块。适合作为前端入门练手、作品集展示或课程设计使用。

## 功能特性
- 响应式页面布局，适配电脑、平板、手机等不同设备
- 完整首页展示模块
- 用户登录页面
- 简洁的视觉设计

## 技术栈
- HTML5
- CSS3

## 项目结构
```
Jingzhe/
├── index.html       # 首页
├── login.html       # 登录页面
├── register.html    # 注册页面
├── css/             # 样式文件
├── images/          # 图片资源
└── README.md        # 项目说明
```

## 快速运行
1. 克隆或下载本项目
2. 直接打开 `index.html` 即可在浏览器中预览

## 项目亮点
- 结构清晰，易于理解和二次修改
- 页面美观，适合作为前端入门作品
- 包含完整的页面体系：首页 + 登录 + 注册
- 纯前端实现，无需后端环境
